# PHASE 73 BEHAVIORAL FLAGS
# Record any outputs that feel confusing, misleading, or misaligned

## FLAG CATEGORIES

### 1. RANKING CONFUSION
- Small input change → large ranking shift
- Large input change → no ranking change  
- Ranking order feels counter-intuitive

### 2. EXPLANATION ISSUES
- Language feels competitive or blaming
- Explanation doesn't match input change
- Tone feels defensive or misleading
- Contains AI/black-box references

### 3. CONFIDENCE SIGNAL PROBLEMS
- High confidence on weak matches
- Low confidence on strong matches  
- Confidence feels unearned or defensive

### 4. FAILURE MODE ISSUES
- User feels blamed for no matches
- No clear next steps provided
- Message tone is negative
